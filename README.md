# shikshaAI Demo — Deployment Guide

## What is this?
A live mobile demo of the shikshaAI Intelligent University Management System,
built for VC-level presentations at Nigerian and Kenyan universities.

## How to deploy to Vercel (15 minutes, free)

### Step 1 — Create a GitHub account
Go to https://github.com and sign up for a free account.

### Step 2 — Upload this project to GitHub
1. Click the green "New" button to create a new repository
2. Name it: shikshaai-demo
3. Click "uploading an existing file"
4. Drag and drop ALL files from this folder into the upload window
5. Click "Commit changes"

### Step 3 — Deploy to Vercel
1. Go to https://vercel.com and sign up with your GitHub account
2. Click "Add New Project"
3. Select the shikshaai-demo repository
4. Click "Deploy" — no settings to change
5. Wait 2 minutes

### Step 4 — Get your link
Vercel gives you a permanent link like:
  https://shikshaai-demo.vercel.app

Send this link via WhatsApp to your colleague in Nigeria.
He opens it in his phone browser — it works like an app.

## Tip — Add to Home Screen
On Android: Open the link in Chrome → tap the 3-dot menu → "Add to Home Screen"
On iPhone: Open in Safari → tap Share → "Add to Home Screen"
The demo will then appear as an app icon on the phone.
